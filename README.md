# Portfolio Insight

Landing page personal de [Maximiliano Yommi](https://www.linkedin.com/in/myommi/) — CEO & Co-Founder de [Cumbre IA](https://cumbre.cloud/).

Glassmorphism dark theme con animación neural mesh de fondo, diseñada para verse en una sola pantalla sin scroll.

## Preview

`https://maxiyommi.github.io/portfolio_insight/`

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

El proyecto se deploya automáticamente a GitHub Pages via GitHub Actions. Cada push a `main` ejecuta el workflow `.github/workflows/deploy.yml`.

### Configuración inicial

1. Crear el repo en GitHub como `portfolio_insight`
2. Ir a **Settings > Pages**
3. En **Source** seleccionar **GitHub Actions**
4. Push a `main`
